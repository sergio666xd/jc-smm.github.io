Volver a inicio

Tipos:

  • Enlaces Internos: los que se dirigen a otras partes dentro de la misma página.

  • Enlaces Locales: los que se dirigen a otras páginas del mismo sitio web.

  • Enlaces Remotos/Externos: los dirigidos hacia páginas de otros sitios web.

  • Enlaces con Direcciones de Correo Electrónico: para crear un mensaje de correo dirigido a una dirección.

  • Enlaces con Número Telefónico: para marcar número telefónico en el teclado movil.

  • Enlaces para Descarga de Archivos: para que los usuarios puedan hacer download de ficheros.
