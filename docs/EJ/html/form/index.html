Formularios

Volver a inicio

Definición:

Un formulario es un conjunto de controles (botones, cajas de texto, casillas de verificación, botones de radio, etc.) que permiten al usuario introducir datos y enviarlos al servidor web para su procesamiento.

La etiqueta que delimita los formularios es <form> ... </form>. Sus atributos más importantes son:

action: contiene el nombre del agente que procesará los datos remitidos al servidor (por ejemplo, un script de PHP).

method: define la manera de enviar los datos al servidor.

Etiquetas:

  • que crean los controles en los formularios

    <input>, <button>, <select>, <optgroup>, <option> y <textarea>.

  • Además se pueden estructurar los controles con las etiquetas <fieldset> y <legend>. Por último la etiqueta <label> permite mejorar la accesibilidad de los controles.

Atributos para elementos:

name: Identifica al control.

type: De la etiqueta <input>, indica el tipo de control de que se trata.

value: Permite establecer el valor inicial de un control, aunque cada control lo utiliza de una forma ligeramente distinta y algunos controles no la admiten (área de texto, archivo o imágen).

id: Identificador del control.
