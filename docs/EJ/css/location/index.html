Ubicación CSS

Volver a inicio

CSS se puede escribir de 3 formas distintas:

  1. En línea:

    Escribir los estilos CSS dentro de la etiqueta por medio del atributo style="", pero solo aplica para la etiqueta que lo contenga. Ejemplo:

    Sin estilo → <p> Texto </p>
    Con estilo →<p style="font-size: 20px;"> Texto </p>

  2. Embebidos:

    Escribir los estilos CSS del documento en general incrustados en el código HTML con la etiqueta <style>, encontrada comúnmente en la cabecera (head). Ejemplo:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
        <style>
            p.emb { font-size: 30px; }
        </style>
    </head>
    <body>
        <p class="emb"> Texto </p>
    </body>
    </html>

  3. CSS Externo:

    La forma más común y más fácil de mantener es utilizando un archivo externo de extensión .css accediendo a él en uno o más documentos HTML, con la etiqueta <link>en la cabecera, de la siguiente forma:

    link rel="stylesheet" href="(ruta del archivo .css)">

    Ejemplo:

    .HTML .CSS
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="stylesheet" href="../styles/location.css">
        <title>Document</title>
    </head>
    <body>
        <p class="emb2"> Texto </p>
    </body>
    </html>
    		 En la ubicación ../styles/location.css
    p.emb2 {
        font-size: 40px;
    }
